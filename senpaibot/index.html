SenpaiBot

A Twitter bot implemented in NodeJS using Twit.

It was a weekend hack, which only really came into being from just playing around with the Twitter API for fun one night.

I was initially playing with the streaming API. Fairly quickly after making it respond to all tweets containing "Notice me Senpai", it began to get a considerable amount of followers. By the time the weekend was over, it had gained a few hundred followers and has continued to gain over 100/day ever since. (When it's not suspended, of course.)

React

React is a game for iOS and Android implemented using Unity.

The game logic and interfacing with various APIs is written in C#. It interfaces with the Google Admob SDK for Unity to display ads, along with both the Apple App Store's and Google Play Store's in-app purchase functionality via the open-source Soomla framework.

GroupQuiz

A custom quizzing site for teachers and students, providing summary statistics about students' quiz sessions to teachers. Built upon MeanJS: MongoDB, Express, AngularJS, and Node.js.

This was made for A-level Computing coursework, which required me to create a software solution for a client using the Waterfall model of software development.
